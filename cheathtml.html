What is HTML?

HTML stands for Hypertext Markup Language. Hypertext is the word or words that contain a link to a website and Markup is the computer language that uses tags to define elements in an HTML document. Html is a programming language used to create and display webpages.

HTML Structure-

    The image shows the basic structure of an HTML document. All Html documents must start with this start code.
    Lets break it down:

  • <!Doctype HTML>:

    • is a declaration on the first line of code in an HTML doc. It defines the document type. This is NOT a tag or element.

  • <html>:

    • tells the browser this is an HTML doc.

  • <head>:

    • element follows the <html> element. This contains the <title> element and metadata.

  • <title>:

  • names the HTML doc. The title will appear on the tab of the webpage.

  • <body>:

    • is where all content on a web page appears.

Elements and tags-

  • Elements:

    • Block-level elements begin on a new line and take up full width available on the screen and can wrap other inline and block-level elements.
    Inline elements render on the same line by default but cannot wrap a block-level element.

  • Tags:

    • Angle brackets = The <> tells the web browser that this is an HTML tag
    The names of the tag usually an abbreviation of a word or phrase. Attributes provide additional information telling the browser what to do to that tag. Attribute values are enclosed in single (‘’) or double (“”) quotes but not all attributes require a value. The content that follows the opening tag is the child of the element. The closing tag has a forward slash (/) before the name of element.

  • Link Elements:

    • Most often used to link to external style sheets.This is an empty element and only contains the sttributes href = the location of the linked document, rel = specifies the relationship between the current doc and the linked doc, and type= specified the media type of the linked doc

HTML <p>, <div>, & Image Elements-

  • <p> elements:

    • The paragraph element allows developers to structure their HTML doc into paragraphs(<pragraph element sturcture>) and have opening and closing tags Paragraph elements are block-level element

  • <div> elements:

    • The div element is the most useable element in web development because it allows them to separate content and data on a webpage into sections. They also have opening and closing tabs and are block-level elements.

  • <Img> elements:

    • Images are added to a webpage using the <img> element. However, it does not have a closing tab, therefore, it can be called an empty tab. It requires 2 attributes-

  • src (soure): points to the location of the image
  • alt (alternative text): provides a description of the image.
