What is HTML?

HTML stands for Hypertext Markup Language. Hypertext is the word or words that contain a link to a website and Markup is the computer language that uses tags to define elements in an HTML document. Html is a programming language used to create and display webpages.

HTML Structure-

1. <!Doctype HTML>:

is a declaration on the first line of code in an HTML doc. It defines the document type. This is NOT a tag or element.

2. <html>:

tells the browser this is an HTML doc.

3. <head>:

element follows the <html> element. This contains the <title> element and metadata.

3. <title>:

names the HTML doc. The title will appear on the tab of the webpage.

4. <body>:

is where all content on a web page appears.
