What is CSS?

CSS stands for Cascading StyleSheet. This allows developers to create rules that specify how the content of an HTML element should appear on the web page. CSS is used to define styles for web pages, including design, layout and variations in display for different devices and screen sizes. It allows HTML to be more content focused by removing the style formatting from HTML page and it also saves a lot of work

CSS Syntax-

CSS works by connecting rules with HTML elements. These rules dictate how the content is displayed by the browser. A single CSS rule is indicated by the opening and closing curly braces {} and contains 2 parts

  • SELECTOR: They indicate which element the rule will apply to. A single rule can apply to many elements by separating the selectors by a comma.
  • DECLARATION: indicate how the element will be styled. All declarations must end with a semicolon ( ; ). Declarations are made up of two parts: Properties and Value. Properties indicate which styling feature or aspect the developer wants to change. Value indicates how the developer wants to change that feature.

CSS Selectors-

Selectors allow you to target or select specific HTML elements and add style. SELECTORS ARE CASE SENSITIVE.
There are various types of selectors:

  • ELEMENT SELECTORS select HTML elements based on their element name.
  • CLASS SELECTORS match an element/elements whose class attribute has a value that matches. When using a class selector in CSS, a period (.) must be placed before the class of the element. An HTML element can have more than one class. Also, more than one element can have the same class value.
  • ID SELECTORS use the id attribute to select specific elements. When selecting an HTML element with an id, a hashtag ( # ) is placed before the element. IDs are unique and can only be used one HTML element.
  • UNIVERSAL SELECTOR uses an asterisk ( * ), targeting all HTML elements on the page.
  • CHILD SELECTORS match the element that is a direct child of another element. Child selectors target elements that are the children of another element but not other elements on the page.
  • DESCENDANT SELECTORS match the element that is a descendant of another specified element. A space between the elements denotes ancestor/descendant relationship.

Stylesheets-

There are three ways CSS can be applied to HTML documents:

  • EXTERNAL STYLESHEETS-
    These are the most preferred way of styling HTML documents. Use one files allows a developer to be more efficient. When creating the CSS style sheets the file must be saved with a .css extension. Each HTML page must include a reference to the external style sheet. This is accomplished by using the <link> element. The <link> element has three attributes:
  • rel- The rel attribute, mandatory, specifies the relationship between the HTML page and the file it is linked to. The value should be stylesheet when linking it to a CSS file.
  • type- This specifies the type of the document being linked to.
  • href- This specifies the path or location to the CSS file. Often the CSS file is placed within a folder called styles.
  • INTERNAL STYLESHEETS-
    These are applied by placing the CSS rules inside of a <style> element, which is nested within the <head> element. The <style> element has opening and closing tags. This method should only be used if a site is being built with one web page.
